# Plan Issues

Use this directory for task-specific refinement notes, blockers, and ambiguity follow-ups.

Guidelines:
- Create one file per task or issue, using the graph node ID in the filename.
- Keep the note idempotent: repeated updates should converge to the same meaning.
- Record the current branch, owned file scope, current progress, and the next intended action.
- If a task needs clarification, include the exact question and the recommended default decision.
- Treat these notes as durable handoff artifacts for a different future agent.
