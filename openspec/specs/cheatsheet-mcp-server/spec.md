# cheatsheet-mcp-server Specification

## Status

deprecated

## Purpose
This spec is a historical tombstone.

The contemplated `tillandsias-cheatsheet-mcp` MCP server is no longer an active
project capability. The live cheatsheet workflow is now owned by:

- `cheatsheet-tooling` for cheatsheet tree structure, template, and generated index
- `cheatsheet-source-layer` / `cheatsheets-license-tiered` for source provenance and verification
- `spec-traceability` for trace graph enforcement

## Notes

- The old MCP tool surface is retained here only for historical traceability.
- Any future MCP-backed cheatsheet query surface must be specified as a new, narrower spec with an executable litmus boundary.
