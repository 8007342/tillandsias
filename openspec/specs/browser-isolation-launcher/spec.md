# browser-isolation-launcher Specification (Tombstone)

## Status

obsolete

## Deprecation Notice

This placeholder spec has been retired. The real browser-launch contract is now
owned by:

- `browser-isolation-core` for the container/orchestration engine
- `browser-isolation-framework` for reusable browser window behavior
- `browser-isolation-tray-integration` for tray-driven launch flows
- `browser-mcp-server` for agent-facing browser requests over MCP

There is no backwards-compatibility commitment.

## Historical Context

This spec existed as a placeholder while the browser isolation stack was being
factored. The launch path has since been split into narrower layers, so this
file now serves only as history.
