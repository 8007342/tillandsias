---
description: Run OpenSpec commands for spec-driven development
---

Run `openspec` CLI commands. OpenSpec manages specifications, changes, and artifact workflows.

Common commands:
- `openspec status` — show current change status
- `openspec list` — list all changes
- `openspec new change "<name>"` — create a new change
- `openspec instructions <artifact> --change "<name>"` — get artifact instructions

OpenSpec is installed at `~/.cache/tillandsias/openspec/bin/openspec`.
